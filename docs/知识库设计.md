# AI模拟面试平台 - 知识库设计

## 1. 知识库概述

### 1.1 核心目标
- 为面试题目提供权威参考答案
- 支持RAG(检索增强生成)提升AI评估准确性
- 为用户提供学习资源推荐
- 支持个性化能力提升建议

### 1.2 技术架构
- **文档存储**: PostgreSQL (元数据) + MinIO/OSS (原始文件)
- **向量存储**: Chroma DB
- **向量化模型**: Sentence Transformers / OpenAI Embeddings
- **RAG框架**: LangChain
- **文档处理**: LangChain Document Loaders + Text Splitters

## 3. 文档处理与向量化流程

### 3.1 文档入库完整流程

```
管理员上传文档
    ↓
1. 文件验证
   - 格式检查(PDF/DOCX/MD/HTML)
   - 大小限制(< 50MB)
   - 病毒扫描
    ↓
2. 保存原始文件
   - 上传到MinIO/OSS
   - 生成唯一文件URL
    ↓
3. 文档解析
   - PDF: PyPDF2 / pdfplumber
   - DOCX: python-docx
   - Markdown: markdown parser
   - HTML: BeautifulSoup
    ↓
4. 文本提取与清洗
   - 去除格式标记
   - 保留结构信息(标题、段落)
   - 去除无意义内容
    ↓
5. 元数据提取
   - 标题、作者、创建时间
   - 自动提取关键词
   - 分类标签
    ↓
6. 存储到PostgreSQL
   - knowledge_documents表
   - 状态: vectorizing
    ↓
7. 文本分块(Chunking)
   - 策略: RecursiveCharacterTextSplitter
   - 块大小: 500字符
   - 重叠: 50字符
   - 保留上下文信息
    ↓
8. 向量化(Embeddings)
   - 模型: sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2
   - 批量处理: 每批32个chunk
    ↓
9. 存储到Chroma DB
   - Collection: knowledge_documents
   - 包含元数据: doc_id, chunk_index, category, tags
    ↓
10. 更新PostgreSQL
    - 更新vectorized状态为true
    - 记录vector_ids
    - 记录chunk_count
    ↓
11. 建立关联
    - 关联到岗位(position_id)
    - 关联到知识点
    ↓
完成入库
```

### 3.2 文档处理代码示例

```python
# services/knowledge_service.py

from langchain.document_loaders import PyPDFLoader, UnstructuredWordDocumentLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter
from langchain.embeddings import HuggingFaceEmbeddings
import chromadb

class KnowledgeService:
    """知识库服务"""
    
    def __init__(self):
        self.text_splitter = RecursiveCharacterTextSplitter(
            chunk_size=500,
            chunk_overlap=50,
            separators=["\n\n", "\n", "。", "!", "?", ",", " ", ""]
        )
        
        self.embeddings = HuggingFaceEmbeddings(
            model_name="sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2"
        )
        
        self.chroma_client = chromadb.Client()
        self.collection = self.chroma_client.get_or_create_collection(
            name="knowledge_documents"
        )
    
    async def process_document(
        self,
        file_path: str,
        file_type: str,
        metadata: dict
    ) -> str:
        """
        处理文档并入库
        
        Args:
            file_path: 文件路径
            file_type: 文件类型 (pdf, docx, markdown)
            metadata: 元数据 (title, category, tags, position_ids)
        
        Returns:
            document_id
        """
        # 1. 加载文档
        documents = await self._load_document(file_path, file_type)
        
        # 2. 文本清洗
        cleaned_text = self._clean_text(documents[0].page_content)
        
        # 3. 保存到PostgreSQL
        doc_id = await self._save_to_db(cleaned_text, metadata)
        
        # 4. 文本分块
        chunks = self.text_splitter.split_text(cleaned_text)
        
        # 5. 向量化并存储
        await self._vectorize_and_store(doc_id, chunks, metadata)
        
        return doc_id
    
    async def _load_document(self, file_path: str, file_type: str):
        """加载文档"""
        if file_type == "pdf":
            loader = PyPDFLoader(file_path)
        elif file_type == "docx":
            loader = UnstructuredWordDocumentLoader(file_path)
        elif file_type == "markdown":
            with open(file_path, 'r', encoding='utf-8') as f:
                content = f.read()
            return [Document(page_content=content)]
        
        return loader.load()
    
    def _clean_text(self, text: str) -> str:
        """文本清洗"""
        # 去除多余空白
        text = re.sub(r'\s+', ' ', text)
        # 去除特殊字符
        text = re.sub(r'[^\w\s\u4e00-\u9fff.,!?;:()（）。，！？；：]', '', text)
        return text.strip()
    
    async def _vectorize_and_store(
        self,
        doc_id: str,
        chunks: list,
        metadata: dict
    ):
        """向量化并存储到Chroma"""
        # 生成向量
        embeddings = self.embeddings.embed_documents(chunks)
        
        # 准备数据
        ids = [f"{doc_id}_chunk_{i}" for i in range(len(chunks))]
        metadatas = [
            {
                "document_id": doc_id,
                "chunk_index": i,
                "category": metadata.get("category", ""),
                "tags": ",".join(metadata.get("tags", [])),
                "position_ids": ",".join(metadata.get("position_ids", []))
            }
            for i in range(len(chunks))
        ]
        
        # 存储到Chroma
        self.collection.add(
            ids=ids,
            embeddings=embeddings,
            documents=chunks,
            metadatas=metadatas
        )
        
        # 更新PostgreSQL
        await self._update_vectorization_status(doc_id, ids, len(chunks))
```

## 4. RAG检索实现

### 4.1 检索策略

**多级检索策略:**

```
用户问题/答案
    ↓
1. 向量检索 (Semantic Search)
   - 将问题向量化
   - Chroma相似度搜索
   - Top-K = 5
    ↓
2. 元数据过滤
   - 按岗位过滤
   - 按分类过滤
   - 按难度过滤
    ↓
3. 重排序 (Re-ranking)
   - 基于相关性分数
   - 基于文档质量分数
   - 基于时效性
    ↓
4. 上下文扩展
   - 获取相邻chunk
   - 补充完整上下文
    ↓
返回Top-3相关文档片段
```

### 4.2 RAG检索代码实现

```python
# services/rag_service.py

class RAGService:
    """RAG检索服务"""
    
    def __init__(self, knowledge_service: KnowledgeService):
        self.knowledge_service = knowledge_service
        self.embeddings = knowledge_service.embeddings
        self.collection = knowledge_service.collection
    
    async def search_knowledge(
        self,
        query: str,
        position_id: str = None,
        category: str = None,
        top_k: int = 5
    ) -> list:
        """
        检索相关知识
        
        Args:
            query: 查询文本(问题或答案)
            position_id: 岗位ID(可选)
            category: 分类(可选)
            top_k: 返回结果数量
        
        Returns:
            [
                {
                    "content": "文档内容",
                    "score": 0.85,
                    "metadata": {...}
                },
                ...
            ]
        """
        # 1. 向量化查询
        query_embedding = self.embeddings.embed_query(query)
        
        # 2. 构建过滤条件
        where_filter = {}
        if position_id:
            where_filter["position_ids"] = {"$contains": position_id}
        if category:
            where_filter["category"] = category
        
        # 3. 向量检索
        results = self.collection.query(
            query_embeddings=[query_embedding],
            n_results=top_k * 2,  # 多取一些用于重排序
            where=where_filter if where_filter else None
        )
        
        # 4. 重排序
        ranked_results = await self._rerank_results(
            results,
            query,
            top_k
        )
        
        # 5. 上下文扩展
        expanded_results = await self._expand_context(ranked_results)
        
        return expanded_results
    
    async def _rerank_results(
        self,
        results: dict,
        query: str,
        top_k: int
    ) -> list:
        """重排序结果"""
        scored_results = []
        
        for i, doc in enumerate(results["documents"][0]):
            metadata = results["metadatas"][0][i]
            distance = results["distances"][0][i]
            
            # 计算综合分数
            similarity_score = 1 - distance  # 转换为相似度
            
            # 获取文档质量分数
            doc_quality = await self._get_document_quality(
                metadata["document_id"]
            )
            
            # 综合评分
            final_score = (
                similarity_score * 0.7 +
                doc_quality * 0.3
            )
            
            scored_results.append({
                "content": doc,
                "score": final_score,
                "metadata": metadata
            })
        
        # 按分数排序
        scored_results.sort(key=lambda x: x["score"], reverse=True)
        
        return scored_results[:top_k]
    
    async def _expand_context(self, results: list) -> list:
        """扩展上下文"""
        expanded = []
        
        for result in results:
            doc_id = result["metadata"]["document_id"]
            chunk_index = result["metadata"]["chunk_index"]
            
            # 获取相邻chunk
            prev_chunk = await self._get_chunk(doc_id, chunk_index - 1)
            next_chunk = await self._get_chunk(doc_id, chunk_index + 1)
            
            # 拼接上下文
            full_context = ""
            if prev_chunk:
                full_context += prev_chunk + "\n"
            full_context += result["content"]
            if next_chunk:
                full_context += "\n" + next_chunk
            
            result["expanded_content"] = full_context
            expanded.append(result)
        
        return expanded
    
    async def generate_answer_with_rag(
        self,
        question: str,
        user_answer: str,
        position_id: str
    ) -> dict:
        """
        基于RAG生成评估参考
        
        Args:
            question: 面试问题
            user_answer: 用户回答
            position_id: 岗位ID
        
        Returns:
            {
                "reference_answer": "参考答案",
                "key_points": ["要点1", "要点2"],
                "sources": [...]
            }
        """
        # 1. 检索相关知识
        knowledge = await self.search_knowledge(
            query=question,
            position_id=position_id,
            top_k=3
        )
        
        # 2. 构建提示词
        context = "\n\n".join([
            f"参考资料{i+1}:\n{k['expanded_content']}"
            for i, k in enumerate(knowledge)
        ])
        
        prompt = f"""
基于以下参考资料,分析用户的回答质量:

面试问题: {question}

参考资料:
{context}

用户回答: {user_answer}

请提供:
1. 标准参考答案
2. 关键要点列表
3. 用户回答的优缺点分析
"""
        
        # 3. 调用LLM生成
        llm_response = await self.llm.generate(prompt)
        
        # 4. 解析响应
        result = self._parse_llm_response(llm_response)
        result["sources"] = [k["metadata"] for k in knowledge]
        
        return result
```

## 7. 知识库质量管理

### 7.1 文档质量评分

```python
def calculate_document_quality(doc: dict) -> float:
    """
    计算文档质量分数
    
    考虑因素:
    - 内容完整性
    - 准确性(人工标注)
    - 时效性
    - 引用次数
    - 用户反馈
    """
    score = 0.0
    
    # 内容完整性 (0-30分)
    completeness = min(len(doc["content"]) / 5000, 1.0) * 30
    score += completeness
    
    # 准确性 (0-30分)
    accuracy = doc.get("accuracy_rating", 0.8) * 30
    score += accuracy
    
    # 时效性 (0-20分)
    age_days = (datetime.now() - doc["created_at"]).days
    timeliness = max(0, 1 - age_days / 365) * 20
    score += timeliness
    
    # 引用次数 (0-10分)
    reference_score = min(doc["reference_count"] / 100, 1.0) * 10
    score += reference_score
    
    # 用户反馈 (0-10分)
    feedback_score = doc.get("avg_rating", 0) * 2
    score += feedback_score
    
    return score / 10  # 归一化到0-10
```
